# Portfolio Setup and Deployment

## Overview
This repository contains the source code for my portfolio and instructions for deploying it on GitHub Pages.

## Prerequisites
- A GitHub account
- Basic knowledge of Git and GitHub
- A web browser

## Steps to Setup the Portfolio
1. **Clone the Repository**
   ```bash
   git clone https://github.com/jeandesirekatihabwa/port.git
   cd port
   ```

2. **Install Required Tools**
   - Make sure you have Node.js installed.
   - You may need a static site generator like Jekyll or any other framework you're using for the portfolio.

3. **Customize Your Portfolio**
   - Open the project in your favorite code editor.
   - Modify the HTML/CSS files or templates as per your need.
   - Add your projects, experiences, and any other relevant information.

## Deploying on GitHub Pages
1. **Push Your Changes to GitHub**
   ```bash
   git add .
   git commit -m "Initial portfolio setup"
   git push origin main
   ```

2. **Enable GitHub Pages**
   - Go to the repository on GitHub.
   - Click on `Settings`.
   - Scroll down to the `Pages` section.
   - Under `Source`, select the `main` branch and `/ (root)` folder.
   - Click `Save`.

3. **Access Your Portfolio**  
   After a few minutes, your portfolio will be available at `https://<your-username>.github.io/<repository-name>/`.

Replace `<your-username>` with your GitHub username and `<repository-name>` with your repository name.

## Conclusion
Your portfolio is now live! Feel free to share the link with potential employers or anyone interested in your work.
